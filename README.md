# House Price Prediction Project

This project predicts house prices using Machine Learning Regression Algorithms.

## Features
- House Price Prediction
- Multiple Regression Models
- Flask Web Application
- Model Evaluation

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy

## Models Used
- Linear Regression
- Random Forest
- KNN
- SVM
- Lasso Regression
- Ridge Regression
